## 三、首页转化优化（最高ROI改进）

### 3.1 Hero区改造

**现状**：纯文字标语 + "Browse Reports →" 按钮

**改造方案**：

```
┌─────────────────────────────────────────────────────────┐
│  [Nav: Reports | Companies | News | Pricing | Get Started]│
├─────────────────────────────────────────────────────────┤
│                                                         │
│   🤖 The AMORA Score  ← 标题左对齐，大字体              │
│   Know which AI, robotics & biotech companies          │
│   will win — before the market catches up.             │
│                                                         │
│   [Start Free →]   [See Scoring Methodology]           │
│   ↑ 主CTA           ↑ 次CTA，降低注册摩擦               │
│                                                         │
│   ─────────────────────────────────────────────────     │
│   Trusted by 500+ analysts & investors                 │
│   [Logo群: 匿名化处理或不放]                             │
│   ─────────────────────────────────────────────────     │
│                                                         │
│   ┌──────────┐  ┌──────────┐  ┌──────────┐            │
│   │  13       │  │  389     │  │  249     │            │
│   │  Reports  │  │  Companies│  │  Articles │            │
│   └──────────┘  └──────────┘  └──────────┘            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**改进要点**：
1. 标题从"Mapping Industries"改为"AMORA Score"为核心 — **差异化一击即中**
2. 主CTA "Start Free"（降低注册摩擦）> "Browse Reports"
3. 次CTA "See Scoring Methodology" — 给犹豫者一个低承诺入口
4. 数字指标放在Hero区下方，增加可信度

### 3.2 导航栏优化

**现状**：Reports | Companies | News | Pricing + 埋得很深的 "Get Started"

**改造方案**：
```
Reports  Companies  News  [▾ Industries ▾]  Pricing
                                          [Start Free] ← 蓝色按钮，常驻
```

### 3.3 注册漏斗重设计（三层钩子）

```
Layer 1: Newsletter ($0，无账户)
  → 入口：首页 /subscribe
  → 价值：每周五 AMORA Weekly，10分钟读完
  → 目标：建立邮件列表，触达即获客

Layer 2: Free Account ($0，需注册)
  → 入口："Start Free" 按钮
  → 价值：访问全部免费报告 + 查看 AMORA Basic Score
  → 目标：建立用户画像，为 Pro 升级埋点

Layer 3: Pro ($29/月 or $149/月 — 待确认定价)
  → 入口：报告内 CTA + Pricing 页
  → 价值：全部报告 + AMORA Full Score + PDF导出
  → 目标：直接收入转化
```

**关键：Layer 2 → Layer 3 的升级路径必须清晰可见**
- 报告详情页加"Pro Only"标记 + "Unlock with Pro →"按钮
- 图表交互锁定 + "Interactive for Pro Members"

## 四、内容策略（上线后持续运营）

### 4.1 爆款内容计划（M4-M6）

**目标**：9个月内 110万次触达，8.8万注册

| 月份 | 内容类型 | 数量 | 目标触达 | 渠道 |
|------|----------|------|----------|------|
| 4月 | 行业快评 + AMORA Weekly | 4篇 + 4期 | 1.5万 | Twitter/X, LinkedIn, Newsletter |
| 5月 | 深度公司分析 + AMORA Weekly | 3篇 + 4期 | 3万（累计） | 同上 + Reddit |
| 6月 | 行业榜单 + AMORA Weekly | 2篇 + 4期 | 5万（累计） | 同上 + YouTube Shorts |

### 4.2 "AMORA Score" 内容IP化

**为什么这是最强获客内容**：

1. **稀缺性**：全球没有第二个深科技公司量化评分系统
2. **可传播**："UBTECH评分8.2 vs 宇树8.9"天然引发讨论
3. **SEO友好**："人形机器人公司排名" = 高搜索量词

**建议内容系列**：
- **"AMORA Top 10"系列**：每赛道TOP10公司评分
  - "Top 10 AI Robotics Companies — AMORA Score Ranking 2026"
  - 标题即传播，内容即获客
- **"本周AMORA Watch"**：3家公司评分变动简评（500字）
- **"AMORA vs Bloomberg"**：用评分对比挑战传统研报（争议性强，传播率高）

### 4.3 社交分享方案（乔治开发中）

| 平台 | 分享格式 | 触发位置 | UTM参数 |
|------|----------|----------|---------|
| Twitter/X | `{URL}?utm_source=twitter&utm_campaign={slug}` | 报告顶部+底部 | `utm_medium=social` |
| LinkedIn | 同上 | 报告顶部+底部 | `utm_source=linkedin` |
| Facebook | `{URL}?utm_source=facebook` | 报告底部 | `utm_medium=social` |
| Reddit | 直接链接 | 手动分享 | `utm_source=reddit` |
| WhatsApp | `https://wa.me/?text={TITLE}%20{URL}` | 报告底部 | - |
| Email | `mailto:?subject={TITLE}&body={SUMMARY}%20{URL}` | 报告底部 | `utm_source=email` |

**分享卡片预览文字（建议在OG标签里配置）**：
```
{title}
AMORA Score: {score}/10 · {industry} sector
{1-sentence summary}
{URL}
```

## 五、定价策略建议（需Franklyn决策）

### 5.1 当前问题

官网显示 Pro = $29/月，但之前内部讨论的是：
- 机构线 Pro = $149/月
- 消费者线 Pro = $19.9/月

**$29 是从哪里来的？这需要立即对齐，否则投资人看到会质疑定价逻辑。**

### 5.2 建议定价结构（基于当前产品成熟度）

| 档位 | 价格 | 内容 | 目标用户 |
|------|------|------|----------|
| **Free** | $0 | 免费报告（3-5篇旗舰报告）+ AMORA Basic Score | 访客、冷启动 |
| **Pro Monthly** | $49/月 | 全部报告 + Full AMORA Score + PDF导出 | 个人投资者、研究员 |
| **Pro Annual** | $399/年（省$189） | 同上 | 价格敏感用户 |
| **Enterprise** | Custom（$800+/月） | API + 定制报告 + 团队协作 | GP、企业战略部门 |

> **注**：$29 → $49 的调整基于：研究工具品类中 $29 低于市场认知价值线（PitchBook最低$125/月），$49 更能支撑"专业研究工具"定位，同时仍低于竞品。
