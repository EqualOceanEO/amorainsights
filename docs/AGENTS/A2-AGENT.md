## 一、核心职责

**Part A2 是整份报告的"结论层"和"变现层"**，是用户付费的核心驱动。负责：

1. **主要玩家财务对比** — 营收/毛利/净利/现金/估值，各公司横向对比
2. **资本效率分析** — 谁用更少的钱做出了更大的事
3. **融资动态追踪** — 近12个月融资事件，估值变化趋势
4. **AMORA Score 评分卡** — 20-25家公司的五维评分（最终权威输出）
5. **投资建议** — 不构成投资建议，但给出结构性机会与风险判断
6. **附录：评分方法论** — 透明披露评分逻辑，供读者自行判断

### A2-5. AMORA Score 企业评分卡 ← 【核心输出】
  （详见第三节评分标准）

### A2-6. 投资建议（免责声明必须前置）
  **声明前置：**
  "以下内容仅供信息参考，不构成投资建议。AMORA 与提及公司无利益关联。"

## 三、AMORA Score 五维评分标准（必须严格遵循）

### 评分维度定义

| 字母 | 维度 | 核心问题 | 评估什么 |
|:---:|------|---------|---------|
| **A** | Advancement | 技术壁垒有多深？ | 专利质量/数量、世界模型进展、数据闭环 |
| **M** | Mastery | 人才优势有多强？ | 核心团队背景、研发人员占比、顶尖人才吸引力 |
| **O** | Operations | 商业落地有多快？ | 收入规模、客户结构、单位经济、部署量 |
| **R** | Reach | 全球化有多广？ | 海外收入占比、海外团队、国际合作 |
| **A** | Affinity | 可持续性有多强？ | 现金储备、融资能力、股东背景、政策友好度 |

### 评分规则（通用，所有行业统一）

```
分值体系：
  10分 = 绝对领先，全球前3，无明显短板
   9分 = 全球前5，有1-2个小短板
   8分 = 行业领先，有若干短板但在可控范围
   7分 = 行业平均偏上，有明显短板
   6分 = 行业平均，短板显著
   5分 = 行业平均偏下，严重依赖外部资源
   4分 = 落后于行业平均，有重大缺陷
   3分 = 边缘玩家，生存压力大
   1-2分 = 基本出局

评分标准：
  ① 每个维度：1段定性判断（≤80字）+ 1个锚点数据
  ② 缺数据 → 标注"数据待补充"，不得硬评
  ③ 附行业均值参照（Top 12 公司的均值）
  ④ 同环比变化：↑（上升）/ ↓（下降）/ →（持平）
```

### 评分卡格式（必须统一）

```
## AMORA Score — {行业名}

| 公司 | A（技术） | M（人才） | O（运营） | R（全球） | A（持续） | 综合 | 行业均值 |
|------|:-------:|:-------:|:-------:|:-------:|:-------:|:----:|:------:|
| 公司A | 8.5 ↑ | 7.0 → | 8.0 ↑ | 9.0 ↑ | 7.5 ↓ | 8.0 | 6.8 |
| 公司B | ... | ... | ... | ... | ... | ... | ... |
| 行业均值 | 6.8 | 6.2 | 5.9 | 6.1 | 6.5 | 6.3 | — |

注：综合分 = (A+M+O+R+A) / 5
    ↑ 过去12个月上升 | ↓ 下降 | → 持平
    * 标注"数据待补充"的维度
```

### 行业校准原则（重要）
- 第一批报告（AI + 人形机器人）完成后，召开校准会
- 校准后锁定评分标准，跨行业保持一致
- 后续报告严格对齐已冻结标准

## 四、数据来源规范

### 财务数据
| 优先级 | 来源 | 可靠性 |
|-------|------|-------|
| ★★★★★ | 上市公司财报（SEC EDGAR / 港交所 / A股年报） | 高 |
| ★★★★★ | 上市公司 IPO 招股书 | 高 |
| ★★★★ | 上市公司业绩会/投资者日材料 | 高 |
| ★★★ | 官方融资公告（金额+轮次+投资方） | 中 |
| ★★ | Crunchbase / LinkedIn（需交叉验证） | 低 |
| ★ | 企业自报数据 | 低 |

### 估值数据
- 上市公司：市值直接引用
- 非上市公司：最新一轮融资估值（注明日期和轮次）
- **严禁使用** PitchBook / CB Insights 未授权数据

## 五、投资建议规范

### 必要免责声明（必须放在最前面，字体不小于正文）

```
【免责声明】
AMORA Score 及本报告中的评分、排名、投资相关内容
仅供信息参考，不构成任何形式的投资建议或荐股。
AMORA 与报告提及的任何公司均无利益关联。
评分基于公开数据，方法论详见附录，不保证完整性或准确性。
投资者应根据自身情况独立判断。
市场有风险，投资需谨慎。
```

### 投资建议内容规范
- ✅ 可以说：细分赛道机会、政策驱动因素、长期趋势
- ✅ 可以说：结构性风险、竞争格局变化、技术颠覆风险
- ❌ 不可说：股价会涨到多少、现在买哪只股票
- ❌ 不可说：某公司"强烈推荐买入"等暗示性语言
