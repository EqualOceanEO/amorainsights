## 一、核心职责

**Part R 是报告的"市场容量层"**，回答"这个市场有多大，谁在全球布局"。负责：

1. **市场规模测算** — TAM/SAM/SOM，分地域、分赛道
2. **三情景预测** — 乐观/基准/保守，附关键假设
3. **细分市场结构** — 各细分市场的增速差异
4. **地域市场分析** — 北美/欧洲/中国/其他亚洲，各市场的独特性
5. **全球化渗透路径** — 从原产地 → 跨国扩张的典型路径
6. **市场渗透率** — 现有规模 vs 潜在规模，渗透率天花板
7. **政策影响评估** — 政府补贴/监管/禁令对市场的影响

## 二、内容标准

### 必含模块（按顺序）

```
## Part R：市场容量与全球化分析

### R-1. 市场规模全景
  - 市场定义（精确边界，与 Part M 的定义一致）
  - 当前市场规模（以美元计，注明基准年）
  - 市场发展历史（5年回顾）
  - 市场规模来源标注（哪家机构，数据年份）

### R-2. TAM / SAM / SOM 拆解
  - TAM（潜在市场总量）：理论最大市场
  - SAM（可服务市场）：AMORA 目标覆盖的市场
  - SOM（可获得市场）：短期内可实际触达的市场
  - 每层拆解：有依据，不是拍脑袋
  - 表格呈现：地域 × 市场层级

### R-3. 三情景预测
  每种情景：
  - 基准情景（Base Case）：中性假设，概率最高（~50%）
  - 乐观情景（Bull Case）：有利假设，概率（~25%）
  - 保守情景（Bear Case）：不利假设，概率（~25%）

  每种情景包含：
    - 市场规模预测（2026-2030 或更长时间）
    - 3个关键假设（用数据支撑）
    - 触发乐观/保守情景的标志性事件

  重要：三种情景都要有，不是只选一个
  三种情景都必须配表格

### R-4. 细分市场结构
  - 各细分赛道的市场规模对比
  - 各细分赛道的增速差异（CAGR）
  - 哪些细分市场增速最快？
  - 哪些细分市场集中度最高？

### R-5. 地域市场深度分析
  每地域：
  - 北美：市场规模 + 增速 + 驱动因素 + 主要玩家
  - 欧洲：市场规模 + 增速 + 驱动因素 + 监管影响
  - 中国：市场规模 + 增速 + 驱动因素 + 补贴政策影响
  - 其他亚洲（除中国）：市场规模 + 增速 + 驱动因素
  - 地域市场对比表格

### R-6. 全球化渗透路径
  - 典型扩张路径：从哪类市场 → 到哪类市场
  - 跨国扩张的主要障碍（监管/本地化/服务网络）
  - 已成功跨国扩张的案例
  - 预测：未来3年哪个地域是下一个增长极？

### R-7. 政策影响评估
  - 主要国家/地区的政策支持（补贴/税收优惠/政府采购）
  - 主要国家/地区的监管限制（安全标准/数据隐私/劳动力法）
  - 政策对市场预测的敏感性分析
  - 中美博弈对该行业市场格局的影响
```

## 三、市场数据规范

### 数据来源优先级
| 优先级 | 来源类型 | 可靠性 |
|-------|---------|-------|
| ★★★★★ | BIS / OECD / 世界银行官方统计数据 | 高 |
| ★★★★ | 上市公司财报中的市场引用（多公司印证） | 高 |
| ★★★★ | 行业协会官方报告（如 IEEE, SAE, 各国机器人协会） | 高 |
| ★★★ | 专业市场研究机构（注明机构名+年份） | 中 |
| ★★ | 新闻引用数据（需找原始来源核实） | 低 |
| ★ | 企业自报市场数据 | 低，不可单独用 |

### 市场预测方法论
**三种情景的假设必须明确：**
```
乐观假设：
  1. {假设} → 来源：{数据}
  2. {假设} → 来源：{数据}
  3. {假设} → 来源：{数据}

基准假设：
  1. {假设} → 来源：{数据}
  2. {假设} → 来源：{数据}
  3. {假设} → 来源：{数据}

保守假设：
  1. {假设} → 来源：{数据}
  2. {假设} → 来源：{数据}
  3. {假设} → 来源：{数据}
```

### CAGR 计算规范
- 必须注明计算起始年和终止年
- 多 CAGR 时必须注明每个 CAGR 的年限
- 格式：`{CAGR}% ({起始年}-{终止年})`
