## AMORA Report Template v1.0 创建（2026-03-28）

建立行业研究报告标准化模板，与企业评估框架 AMORA Score 并行：

**五维框架：**
- **A** - Advancement（技术先进性）：技术壁垒、路线分歧、卡脖子分析、**世界模型**
- **M** - Mapping（产业链生态位）：上下游分布、中美对比、空白机会
- **O** - Operations（商业化运营）：客户结构、ROI、落地案例
- **R** - Reach（市场容量）：规模预测、方法论、三情景
- **A** - Assets（资本价值）：财务对比、估值排名、资本效率

**核心原则：**
- 框架固化（五维字母+章节顺序）→ 建立品牌识别
- 内容灵活（子章节+数据呈现）→ 保持行业针对性
- 每章必须有"核心发现"段 + 中美对比结论 + So What

- 评估五维度：环境理解、物理预测、规划决策、仿真训练、数据引擎
