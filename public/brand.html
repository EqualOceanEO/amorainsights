Usage Rules
✓ DO Use on dark backgrounds (#060D1C or darker)
Keep minimum clear space = icon height on all sides
Use SVG source files only
Maintain original proportions
✗ DON'T Don't place on busy or patterned backgrounds
Don't stretch or distort the logo
Don't change the gradient colors
Don't add drop shadows or effects
✓ DO Use icon-only mark for favicon, app icon, avatar
Use horizontal lockup for nav bars and headers
Minimum size: 16px for icon, 120px wide for lockup
✗ DON'T Don't use rasterized PNG when SVG is available
Don't recreate the logo in other tools
Don't use on mid-gray backgrounds
